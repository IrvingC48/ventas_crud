Inventario de ventas


Página diseñada para registrar pedidos y productos, utilizando el modelo CRUD (Create, Reade, Update, Delete) con JS, y almacenando los datos en el Local Storage.



Contenido:


  • CREAR VENTA: Aquí podemos iniciar un nuevo pedido, y agregar los productos que desea el cliente.
  • PRODUCTOS: Dentro de este apartado, encontraremos la sección del catálogo de productos que se venden. Se pueden agregar, modificar o eliminar los productos.
  • VENTAS TOTALES: Dentro de una tabla, se visualizan las ventas que se han registrado en la aplicación.
